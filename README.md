## Overview

This document outlines a comprehensive personal AI fluency development plan based on the Ai Fluency Framework Foundations course. It includes learning objectives, milestones, resources, assignments, reflection practices, and peer review mechanisms to ensure structured progress toward AI fluency.

## 1. Learning Objectives & Key Competencies

| Competency | Description |
|------------|-------------|
| **Data Literacy** | Ability to collect, clean, analyze, and interpret data relevant to AI projects. |
| **Model Understanding** | Knowledge of model architectures, training processes, evaluation metrics, and deployment considerations. |
| **Ethical & Responsible AI** | Awareness of bias, fairness, transparency, privacy, and societal impact in AI systems. |
| **Hands‑On Engineering** | Practical skills in building, fine‑tuning, and deploying AI models (e.g., chatbots). |
| **Critical Evaluation** | Capability to assess model outputs, detect hallucinations, and validate results. |
| **Communication & Collaboration** | Effectively explain AI concepts to non‑technical stakeholders and work within interdisciplinary teams. |

## 4. Practical Assignments

1. **Data Cleaning Notebook** – Import a public dataset, handle missing values, and visualize distributions.
2. **Model Building** – Train a regression/classification model; report accuracy & confusion matrix.
3. **Bias Essay** – Analyze a dataset for demographic bias; propose mitigation steps.
4. **Chatbot Prototype** – Use Rasa or LangChain to create a simple FAQ chatbot.
5. **Hallucination Evaluation** – Generate responses, flag hallucinations, and suggest improvements.
